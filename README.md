# RAG_QA for 海外旅行不便險條款

本專案實作一個 **檢索增強生成 (RAG)** 系統，針對《海外旅行不便險條款》PDF 提供自動化問答。  
使用者可輸入自然語言問題，系統會檢索相關條文並生成結構化回答。

---

## 📂 專案架構

- `rag_travel.py`  
  主程式，負責 PDF 解析、檢索、主題路由、答案生成。
- `request.txt`  
  所需套件清單，可用於安裝環境。
- `海外旅行不便險條款.pdf`  
  測試用保險條款文件（需自行提供）。
